## Теория

Граф — структура данных из вершин и рёбер. В данной работе реализован **взвешенный неориентированный граф** на основе матрицы смежности.

Алгоритм **Флойда–Уоршелла** находит кратчайшие пути между всеми парами вершин за O(n³):

```
for k in 0..n:
  for i in 0..n:
    for j in 0..n:
      if dist[i][k] + dist[k][j] < dist[i][j]:
        dist[i][j] = dist[i][k] + dist[k][j]
```

| Операция               | Сложность |
|------------------------|-----------|
| Создание графа         | O(n²)     |
| Добавление ребра       | O(1)      |
| Удаление ребра         | O(1)      |
| Поиск веса ребра       | O(1)      |
| Floyd-Warshall         | O(n³)     |
| Память                 | O(n²)     |

## Структуры данных

Граф хранится как матрица смежности размером n×n. Отсутствие ребра обозначается `INF = INT_MAX / 2`.

```c
typedef struct {
    int  n;
    int **matrix;
} Graph;
```

## Алгоритм Floyd-Warshall

Инициализируется матрица расстояний из матрицы смежности, затем тройным циклом по промежуточным вершинам k улучшаются пути `i → k → j`.

## Запуск из CLI

```
lab5.exe --load <input.txt> --result <result.txt> [--save-graph <out.txt>] [--src N --dst N]
```

| Флаг                  | Описание                                      |
|-----------------------|-----------------------------------------------|
| `--load <file>`       | загрузить граф из файла (обязательно)         |
| `--result <file>`     | сохранить матрицы расстояний и путей (обязательно) |
| `--save-graph <file>` | пересохранить граф в файл (опционально)       |
| `--src N --dst N`     | вывести кратчайший путь между двумя вершинами |
| `--help`              | справка                                       |

Примеры:
```
lab5.exe --load sample.txt --result result.txt
lab5.exe --load sample.txt --result result.txt --src 0 --dst 3
lab5.exe --load sample.txt --result result.txt --save-graph copy.txt
```

## Вывод

Реализован взвешенный неориентированный граф на матрице смежности. Алгоритм Флойда–Уоршелла находит кратчайшие пути между всеми парами вершин за O(n³) и восстанавливает конкретные маршруты через матрицу предшественников.
