## Описание

В работе написан Bash-скрипт `9_var.sh`, который обрабатывает файлы в выбранном каталоге.
Скрипт ищет файлы с указанным суффиксом и, если основная часть имени короче заданной длины,
дополняет её выбранным символом.

Например, файл `test.txt` при суффиксе `.txt`, длине `8` и символе `_`
будет переименован в `test____.txt`.

---

## Используемые возможности Bash

- проверка количества аргументов командной строки;
- работа с переменными и параметрами скрипта;
- проверка существования каталога и файлов;
- цикл `for` для перебора файлов;
- строковые операции для выделения имени и суффикса;
- арифметические выражения для расчёта количества символов;
- команда `mv` для переименования файлов.

---

## Запуск

```bash
./9_var.sh <суффикс> <длина> <символ> [каталог]
```

Параметры:

| Параметр | Описание |
|----------|----------|
| `суффикс` | окончание файлов, которые нужно обработать |
| `длина` | нужная длина имени без суффикса |
| `символ` | символ, которым дополняется имя |
| `каталог` | папка для обработки; если не указана, используется текущая |

Пример:

```bash
./9_var.sh .txt 10 _ ./files
```
